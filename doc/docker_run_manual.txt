1. Abra la terminal y digite:

	$ git clone https://github.com/josepabloapu/IE-0724-Proyecto-1.git

2. Dirijase al directorio del proyecto clonado y digite en la terminal:
	
	$ docker build --tag rb_tree_G5:1.0 .

3. Digite ahora lo siguiente para correr las pruebas unitarias del proyecto:
	
	$ docker run -ti rb_tree_G5:1.0 /usr/bin/run_unittest
